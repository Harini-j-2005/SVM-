# Text Classification using Support Vector Machines (SVM)

This project implements a **text classification system** using **Support Vector Machines (SVM)** and **TF-IDF feature extraction**.  
The trained model is deployed as a **Flask web application** that classifies user-input text into predefined categories in real time.
